# Build Resources

This folder contains build resources for electron-builder.

Place the following files here for customization:
- `icon.ico` - Windows app icon (256x256 or larger)
- `icon.icns` - macOS app icon
- `icon.png` - Linux app icon (512x512)
- `background.png` - DMG background (macOS)
- `installerSidebar.bmp` - NSIS installer sidebar (164x314)
